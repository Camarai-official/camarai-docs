# 🗄️ Base de Datos Maestro (Convex)

Este documento es la referencia técnica central para el esquema de base de datos. Todas las tablas están en formato de rejilla para facilitar la visualización de columnas, tipos de datos e índices.

---

## 🏢 1. Configuración y Empresa

### Tabla: `establishments`
*Datos del local y configuración global.*

| Columna | Tipo | Index | FK | Descripción |
|---|---|---|---|---|
| `_id` | `Id<"establishments">` |**Yes** | - | Identificador único del local. |
| `nombre` | `string` | No | - | Nombre comercial del negocio. |
| `cif` | `string` |**Yes** | - | Identificador fiscal único. |
| `telefono` | `string` | No | - | Teléfono de contacto. |
| `email` | `string` | No | - | Email corporativo. |
| `direccion` | `string` | No | - | Dirección física. |
| `logo_url` | `string` | No | - | URL de imagen en Convex Storage. |
| `timezone` | `string` | No | - | Zona horaria (ej: "Europe/Madrid"). |
| `currency` | `string` | No | - | Símbolo de moneda (ej: "EUR"). |
| `owner_id` | `Id<"users">` |**Yes** | `users` | Referencia al dueño de la cuenta. |
| `plan` | `string` | No | - | "free", "starter", "pro", "enterprise". |
| `payment_methods` | `object` | No | - | `{ cash: bool, card: bool, bizum: bool, mixed: bool }` |
