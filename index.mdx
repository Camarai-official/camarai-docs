# Documentación Camarai

Bienvenido a la documentación de **Camarai**, el sistema de gestión integral para restaurantes.

## Comenzar

Explora las secciones del menú lateral para conocer todas las funcionalidades del sistema:

- **General**: Introducción al sistema
- **Ingeniería**: Base de datos y auditoría
- **Operativa**: Dashboard, comandas, POS, plano de mesas y ambientes
- **Gestión**: Inventario, cartas, reservas, personal y reportes
- **Marketing y Clientes**: Gestión de clientes, promociones y notificaciones
- **Configuración**: Ajustes del sistema

¿Prefieres empezar por algún lugar específico? Puedes acceder directamente a cualquier sección desde la barra lateral.
