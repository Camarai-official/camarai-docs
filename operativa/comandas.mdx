  <Tab title="Historial de Pedidos">
    ## ¿Qué es esta vista?
    Las Comandas son **el registro de cada pedido** creado en el sistema. Es el libro mayor de auditoría: permite buscar, filtrar y exportar cualquier venta.

    La tabla muestra páginas de **10 pedidos** con las siguientes columnas principales:

    | Columna | Editable | Datos DB |
    |---|---|---|
    | **Nº Orden** | No | `order_number` |
    | **Mesa** | Sí | `table_id` |
    | **Nombre** | Sí | `client_name` |
    | **Total** | No directo | Calculado de `order_items` |
    | **Estado** | Sí | `status` |
  </Tab>

  <Tab title="Edición">
    ## 🪟 Modal: Editar Comanda
    ![Modal Editar Comanda](/images/modal-comandas-editar.png)

    Dividido en dos columnas para máxima velocidad:
    - **Izquierda**: Gestión de productos (buscador + cantidades +/-).
    - **Derecha**: Detalles del servicio (Mesa, Cliente, Notas de cocina, Descuento %).

    <Note icon="floppy-disk">
      Al guardar, se actualiza el documento en la colección `orders` y se recalculan los `order_items` relacionados.
    </Note>
  </Tab>
