# Chat Interno

El Dashboard incluye un chat integrado para que los administradores, camareros (vía POS) y gerentes puedan comunicarse sin salir de la plataforma.

## Características

- **Mensajería Instantánea**: Comunicación directa entre dispositivos Camelai.
- **Canales por Ambiente**: Grupos automáticos por zona de trabajo.
- **Integración con Agentes**: Los agentes pueden enviar alertas críticas directamente al chat.
