  <Tab title="Gestión de Menú">
    ## ¿Qué es esta vista?
    Esta vista agrupa en un solo lugar las **3 capas del menú**: la Carta (el documento), las Categorías (apartados) y los Productos (platos individuales).

    La jerarquía es: `Carta → Categoría → Producto`.

    | Nivel | Descripción | Tabla Convex |
    | :--- | :--- | :--- |
    | **Carta** | El menú global (Verano, Vinos, etc.) | `menus` |
    | **Categoría** | Entrantes, Carnes, Postres, etc. | `categories` |
    | **Producto** | El plato o bebida individual con su precio. | `products` |

    ### 🪟 Modal: Editar Categoría
    ![Modal Categoría](/images/cartas-modal-categoria.png)
    Permite cambiar el nombre, asignar a una carta distinta y subir un icono específico.
  </Tab>

    <Note icon="triangle-exclamation" color="#FF0000">
      **🚨 HAY QUE SOLUCIONAR (CRÍTICO)**: Actualmente existe una desalineación lingüística. El Dashboard usa `nombre_producto` y la Carta/POS usan `name`.

      **🚨 HAY QUE SOLUCIONAR**: Los precios se están guardando como strings con símbolo (`'20.97€'`). 
      **Acción**: Deben ser `number` puros para permitir cálculos automáticos de IVA y descuentos.

    ### 🛠️ Mapeo de Sincronización
    | Campo | Dashboard (ES ⚠️) | Estándar Convex (EN ✅) |
    | :--- | :--- | :--- |
    | Nombre | `nombre_producto` | `name` |
    | Precio | `precio_venta` | `price` |
    | Imagen | `url_imagen_producto` | `image` |
    | Estado | `'Activo' / 'Inactivo'` | `active: boolean` |
