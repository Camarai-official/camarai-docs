    ### Secciones del Modal
    - **General**: Nombre, imagen (128×128), categoría y descripción.
    - **Precios**: PVP, impuesto aplicable y cálculo de rentabilidad (Margen Bruto).
    - **Receta**: Escandallo vinculado a `ingredients` para coste de producción.
    - **Variantes**: Modificadores y extras opcionales (ej: "Grande" +1.50€).
    - **Disponibilidad**: Horario, visibilidad y checkboxes de alérgenos (Gluten, Crustáceos, Huevos, Pescado, Cacahuetes, Soja).
  </Tab>

    ### Esquema Convex: `products`
    | Campo | Tipo | Estado |
    | :--- | :--- | :--- |
    | `name` | `string` | ✅ Estándar |
    | `price` | `number` | ✅ Estándar (céntimos) |
    | `category_id` | `Id<"categories">` | ✅ Relacional |
    | `active` | `boolean` | ✅ Estado |
    | `image` | `string` (URL) | ✅ Convex Storage |
    | `description` | `string` | ✅ Descripción |
    | `allergens` | `string[]` | ✅ Lista de alérgenos |
    | `variants` | `array` | ✅ Modificadores |
  </Tab>
