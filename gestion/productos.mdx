# Productos

En esta sección se definen todos los elementos que componen la oferta del restaurante. Es el core de la base de datos de ventas.

## Gestión de Ficha Técnica

- **Información Básica**: Nombre, descripción, imágenes.
- **Precios y Costes**: Gestión de márgenes.
- **Impuestos**: Vinculación con tipos de IVA.
- **Variantes y Modificadores**: Opciones de personalización.
- **Alérgenos**: Clasificación para la Carta Digital.
